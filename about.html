About the Project

Built with
Passion & Precision

FinCal Electric is a love letter to beautiful, functional web applications. No frameworks. No build tools. Just clean code and great design.

Why FinCal Electric?

Most financial calculators online look like they were built in 2005 and never updated. They get the job done, but they don't make you want to explore your financial future.

We wanted to change that. FinCal Electric combines the depth of a scientific calculator with the intelligence of a financial planner — wrapped in a design that feels alive. Every gradient, every animation, every micro-interaction is crafted by hand.

The result? A tool that doesn't just calculate numbers — it helps you understand and visualize your money's journey.

⚡
What's Inside

Feature Highlights

Everything you need for number crunching, all running in your browser with zero server calls.

🧮

Scientific Calculator

Trigonometry, logarithms, factorials, powers, constants (π, e), memory operations — everything a scientific calculator should have.

📊

Live Visualizations

Interactive Chart.js graphs that update in real-time as you adjust sliders. See your growth trajectory and portfolio mix at a glance.

🌓

Dual Themes

A warm light mode with sunrise gradients and a sleek dark mode with deep indigo tones. Each theme is fully custom-designed.

🔗

Calc ↔ Finance Bridge

Transfer results from the scientific calculator directly into financial sliders. Click stat cards to load values back. Seamless workflow.

📱

Fully Responsive

Works beautifully on desktop, tablet, and mobile. The three-column layout gracefully collapses into a single column on smaller screens.

⚡

Zero Dependencies

No React, no Webpack, no npm install. Just HTML, CSS, and JavaScript with Chart.js and Font Awesome via CDN. Open and go.
